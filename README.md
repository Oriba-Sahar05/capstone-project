# Capstone Project:

A capstone repository for the AI-assisted development track. This project will evolve into a full-stack application built with Node.js, developed incrementally with AI tooling (Cursor) and a disciplined git workflow.

## Goals:

- Establish a reproducible development environment (Node.js LTS, Git, Cursor)
- Follow [Conventional Commits](https://www.conventionalcommits.org/) for all changes
- Use AI assistants for code, documentation, and review while maintaining clear project conventions

## Stack: (planned)

- **Runtime:** Node.js (LTS)
- **Package manager:** npm
- **Editor / AI:** Cursor
- **Version control:** Git + GitHub

## Status:

🚧 **Setup phase** — repository scaffold, license, and project conventions are in place. Application features will be added in later phases.
